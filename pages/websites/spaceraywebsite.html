About


SpaceRay's main website had just had a freshening up before I was put on the project. Despite this I still had work to do adding in pages, such as the Coldblocker page shown below, creating galleries, adding documents and doing further tweaking to existing pages. When adding pages extra attention to detail was paid to make sure that the new pages matched the current aesthetic the page already had.

FAST FACTS


In order to make the site accessible, SpaceRay's site was built with the Wordpress CMS. Special styling and more complex coding was done with HTML5 and CSS and small tweaks to the template's php to ensure better stability when templates were updated. Special graphics were done in illustrator and images manipulated in Photoshop CC.
